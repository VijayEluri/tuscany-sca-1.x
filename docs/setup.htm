Tuscany Sample Setup

Requirements

The only requirement to run the samples is a Java 5.0 JRE. If you don't already  have this you can download it at:   http://java.sun.com/j2se/1.5.0/download.jsp

Setup

The only  requirement for the web based demos is that you have the tomcat server in your distribution running:
  • Change the directory to  apache-tomcat-5.5.17\bin
  • Execute: startup
