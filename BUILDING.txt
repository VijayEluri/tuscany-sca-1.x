Building tuscany/java
=====================

Initial Setup
-------------

1) Install J2SE 5.0 SDK (1.5.2_06 works)

2) Install Maven:
Download maven 2.0.4 (download from http://maven.apache.org/download.html).
Set the env variable MVN_HOME to point at your maven installation.
Add $MVN_HOME/bin to your path.


Building
--------

1) cd tuscany/java
2) mvn
