Installation

This page describes what is needed to install and run Tuscany SCA, SDO and DAS for Java.
If you haven't already, the first step is to download the latest release of apache tuscany from our downloads page.

System Requirements

In order to run the Tuscany there are some minimum requirements

Software Download Link
Operating systems:
  • windows xp sp2
  • linux
JDK 5.0 http://java.sun.com/j2se/1.5.0/download.jsp
Apache Maven 2.0.4
(Optional: Only if you want to build the samples yourself) 		http://maven.apache.org/download.html

Bulding Samples

The samples are already built and ready to run. However if you want to build them yourself you can using Maven.
For example to build the helloworld sample:

  • Change to the directory: samples/sca/helloworld dirctory
  • Issue the command: mvn

Starting the Server

Some of the samples require the Tuscany embeded Tomcat server to be started.

  • Change the directory to apache-tomcat-5.5.17\bin
  • Issue the command: startup
  • To later stop the server Issue the command: shutdown
