Overview

This sample illustrates the use of SCA components implemented using composites. I.e. it demonstrates how composites can be recusively composed using SCA

Sample Structure

Location

This sample is located  in the samples\composite-impl directory. All the following commands should be issued while working in the sample directory.

Prerequisites

Obtain the following prerequisites and install according to their documentation.

  • JDK 5.0
  • Maven 2.0.4

Building

To build the sample  issue :
