Tuscany Calculator RMI Service Binding Sample

Overview

This sample illustrates the use of Tuscany RMI Binding to expose component services as Java RMI Services. It also demonstrates how a simple Java RMI Client application invoke the SCA Application's services exposed as RMI Services

About the sample

This sample contains the Calculator SCA Application defined in the CalculatorRMIServer.composite. This composite exposes the Calculator functions as RMI Services.

Besides the classes that provide the basic calculator function, there are two other classes that are key to running this sample. They are : -

 - CalculatorServer.java : This class starts the SCA Runtime and deploys the CalculatorRMIServer.composite. Running this class keeps the SCA Runtime with the deployed application up an running till the 'Enter Key' is pressed on the console.

 - CalculatorClient.java : This is simple Java RMI calculator client that will invoke the Calculator SCA Appplication's funtion using Java RMI

Running the sample

- First run the CalculatorServer class. This clss will start the SCA Runtime and deploy the CalculatorRMIServer.composite. The SCA Runtime will be kept up and running until the 'Enter Key' is pressed on the console.

- Next run the CalculatorClient class. This class will invoke the service provided by the CalculatorServer using Java RMI. Running this class successfully will display the following to the standard output:
3 + 2=5.0
3 - 2=1.0
3 * 2=6.0
3 / 2=1.5

Now go over to the console running the CalculatorServer and press the 'Enter' key to end that application.
