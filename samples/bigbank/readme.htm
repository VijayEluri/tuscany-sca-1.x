Tuscany BigBank Sample Application.

Overview

The Tuscany BigBank sample applications  shows using the Tuscany SCA runtime to build an actual financial application.

Setup

Set up your server as described in the Tomcat Setup.  If you are not configuring with the acceptance test the server, you will need to add
samples\bigbank\account\target\bigbank-account-SNAPSHOT.war and amples\bigbank\webclient\target\bigbank-webclient-SNAPSHOT.war  to your tomcat's webapp directory.

Running

Once you've started your web server, use your browse to view the following URL: http://localhost:8080/bigbank-webclient-SNAPSHOT/login.html
