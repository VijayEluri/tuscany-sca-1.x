Tuscany Calculator-Combo - A Simple SCA Assembly that hosts calculator functions as WebServices

Overview

This sample is an assembly of Calculator functions implemented in Java and exposed as Web Services. The primary objective of developing this sample is to support the Calculator-Combo sample.

To test this sample you might have to either run the Calculator-Combo sample after running this sample or you might right your won SCA Application that refereences the services of this sample

Location

This sample is located in the following path of the samples source distribution: -
<source disb>/samples/webapp/calculatorws

After you build the sample source the deployable 'war' can be found in the 'target' directory of the the project as::
<source disb>/samples/webapp/calculatorws/target/sample-calculatorws.war

If there is a binary distribution of the samples available then you may just about look for the sample-calculatorws.war in the distribution.

Setup

1) Install the Apache Tomcat Server (latest release) and copy the sample-calculatorws.war into the WebApps directory of the Tomcat Installation.

Running

Start the Apache Tomcat Server and ensure that the Calculator WebService is deployed and is running.

To test this sample completely, run the Calculator-Combo sample after running this sample. For details on setting up and running the Calculator-Combo sample look at the Calculator-Combo-Readme.htm there.
