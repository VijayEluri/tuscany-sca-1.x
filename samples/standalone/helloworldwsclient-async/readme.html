Tuscany Hello World Async Client Sample

Overview

The Tuscany helloworld sample shows using the Tuscany SCA runtime in a J2SE environment  loading a component that implements asynchronous web service call.

Location

This sample is located  in the samples\standalone\helloworldwsclient-async directory. All the following commands should be issued while working in the sample directory.

Building

Obtain  JDK 5.0 and Maven 2.0.4 and set up according to their documentation.
To build issue :
mvn

Setup

This sample depends on the Tuscany Hello World WS sample already running to execute successfully
Set up the standalone runtime environment using the following:
mvn dependency:unpack

Running

Execute the following command: (cut and paste to command line)

java -jar target\distribution\bin\launcher.jar target\sample-helloworldwsclient-async-1.0-incubator-M2.jar

Results

The sample when run should simply display to the standard output:
Hello World
